Get up and running with typed-xlsx to create your first Excel export in just a few minutes.

<Note>
  Make sure you have Node.js 16+ installed and typed-xlsx added to your project. See the [installation guide](/installation) if you haven't installed it yet.
</Note>

## Overview

Creating an Excel export with typed-xlsx follows three simple steps:

<Steps>
  <Step title="Define your data types">
    Define TypeScript interfaces for your data
  </Step>
  <Step title="Build a schema">
    Create a type-safe schema using ExcelSchemaBuilder
  </Step>
  <Step title="Generate the Excel file">
    Use ExcelBuilder to create the workbook
  </Step>
</Steps>

## Step 1: Define your data types

First, define the TypeScript interface for your data:

```typescript
interface User {
  id: number
  firstName: string
  lastName: string
  email: string
  balance: number
  createdAt: Date
}
```

## Step 2: Build a schema

Create a schema that maps your data to Excel columns:

```typescript
import { ExcelSchemaBuilder } from '@chronicstone/typed-xlsx'

const userSchema = ExcelSchemaBuilder
  .create<User>()
  .column('id', { 
    key: 'id',
    label: 'ID'
  })
  .column('firstName', { 
    key: 'firstName',
    label: 'First Name'
  })
  .column('lastName', { 
    key: 'lastName',
    label: 'Last Name'
  })
  .column('email', { 
    key: 'email',
    label: 'Email'
  })
  .column('balance', {
    key: 'balance',
    label: 'Balance',
    format: '"$"#,##0.00'
  })
  .column('createdAt', {
    key: 'createdAt',
    label: 'Created',
    format: 'd mmm yyyy'
  })
  .build()
```

<Tip>
  TypeScript will provide autocomplete for column keys based on your data type, and catch typos at compile time!
</Tip>

## Step 3: Generate the Excel file

Use ExcelBuilder to create your workbook:

```typescript
import { ExcelBuilder } from '@chronicstone/typed-xlsx'
import fs from 'fs'

const users: User[] = [
  {
    id: 1,
    firstName: 'John',
    lastName: 'Doe',
    email: 'john.doe@example.com',
    balance: 1250.50,
    createdAt: new Date('2024-01-15')
  },
  {
    id: 2,
    firstName: 'Jane',
    lastName: 'Smith',
    email: 'jane.smith@example.com',
    balance: 3420.75,
    createdAt: new Date('2024-02-20')
  }
]

const buffer = ExcelBuilder
  .create()
  .sheet('Users')
  .addTable({
    data: users,
    schema: userSchema
  })
  .build({ output: 'buffer' })

// Save to file
fs.writeFileSync('users.xlsx', buffer)
```

That's it! You now have a fully formatted Excel file with your user data.

## Adding transformations

You can transform data before it's written to Excel:

```typescript
interface User {
  id: number
  firstName: string
  lastName: string
  email: string
  roles: string[]
  isActive: boolean
}

// Define reusable transformers
const transformers = {
  list: (value: string[]) => value.join(', '),
  boolean: (value: boolean) => value ? 'Yes' : 'No'
}

const userSchema = ExcelSchemaBuilder
  .create<User>()
  .withTransformers(transformers)
  .column('id', { key: 'id' })
  .column('firstName', { key: 'firstName' })
  .column('lastName', { key: 'lastName' })
  .column('email', { key: 'email' })
  .column('roles', {
    key: 'roles',
    transform: 'list' // Use the named transformer
  })
  .column('status', {
    key: 'isActive',
    transform: 'boolean'
  })
  .build()
```

## Adding summaries

Add summary rows to calculate totals, averages, and more:

```typescript
const schema = ExcelSchemaBuilder
  .create<User>()
  .column('id', {
    key: 'id',
    summary: [{
      value: () => 'TOTAL'
    }]
  })
  .column('balance', {
    key: 'balance',
    format: '"$"#,##0.00',
    summary: [{
      value: data => data.reduce((sum, user) => sum + user.balance, 0),
      format: '"$"#,##0.00'
    }]
  })
  .build()
```
